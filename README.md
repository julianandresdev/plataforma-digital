# Servicios Comunitarios - Fundación Mi Oportunidad

Plataforma web para conectar vecinos de sectores en Cali, Colombia, facilitando el intercambio de servicios comunitarios.

## Características

- Sistema de login simple
- Publicación de servicios ofrecidos
- Solicitudes de servicios necesarios
- Filtros por categoría y urgencia
- Almacenamiento local con LocalStorage
- Diseño responsive mobile-first

## Tecnologías

- HTML5
- CSS3
- JavaScript Vanilla
- Bootstrap 5
- LocalStorage

## Despliegue

Este proyecto está diseñado para desplegarse en Vercel sin configuración adicional.

## Instalación Local

1. Clona el repositorio
2. Abre `index.html` en tu navegador
